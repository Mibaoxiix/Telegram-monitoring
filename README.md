# 📡 Telegram 消息监听器

一个基于 Python  的 Telegram 客户端工具，支持消息监听、规则过滤、自适应界面

---

## ✨ 功能特性

* ✅ Telegram 登录（验证码 / 2FA）
* ✅ 实时监听群组 / 私聊消息
* ✅ 自定义关键词过滤
* ✅ 多规则监听
* ✅ 自适应布局（兼容小窗口 / 全屏）
* ✅ 简约现代 UI（卡片风格）

---

## 🔑 登录方式

* 📱 手机号 + 验证码登录
* 🔐 支持 Telegram 二步验证（2FA）

---

## 📂 项目结构

```
.
├── main.py                # 主程序入口
├── sessions/             # 登录会话文件
├── config.json           # 配置文件（可选）
└── README.md             # 项目说明
```

---

## 🧠 核心功能说明

### 📥 消息监听

* 支持监听群组 / 私聊
* 自动实时更新消息列表
* 可结合关键词过滤

---

### 🎯 规则系统

* 支持关键词匹配
* 支持多规则组合
* 可扩展自动操作（如转发/通知）

---

### 🧩 自适应布局

* 窗口缩小时自动切换布局
* 三栏 → 上下结构
* 顶部按钮自动换行
